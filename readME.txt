Simple-Classroom

Built with MeteorJS - A simple prototype full-stack application incorporating digital ui-design and practice modules in the Meteor Framework.


Application features:

    A simple direct chat with role-assigned labels in a chat channel.
    A simple multiple choice quiz section.
    A simple survey "click to see highest count" types.
    A simple forum with vote up/vote down posts concept.
    A simple file storage concept.
    A simple HTML-Jquery collaborative drawing canvas.
