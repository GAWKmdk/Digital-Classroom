Simple-Classroom

Built with MeteorJS - A simple prototype incorporating digital teaching features.

Application features:

    direct chat with role-assigned labels in a chat channel.
    quiz-style questions with multiple choice or survey "click to count" types.
    a forum like Meteorite (panel cards with discuss buttons).
    

Currently in-build

    
    collaborative drawing board
    user accounts built around twitter login
